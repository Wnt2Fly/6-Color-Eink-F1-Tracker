# README hero image

Save your device photo here as **`readme-photo.jpg`** (JPEG recommended, ~1200–1872 px wide).

Google Photos share links (e.g. `photos.fife.usercontent.google.com/...`) require a Google login and **do not work** in GitHub READMEs. Export from Google Photos:

1. Open the photo → **Share** or **Download**.
2. Save/download as `readme-photo.jpg` into this folder.
3. Commit and push — the image appears at the top of the root `README.md`.

Optional: use PNG and change the path in `README.md` to `images/readme-photo.png`.
